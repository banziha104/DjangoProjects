## Git Deploy

<li> git init
<li> git config --global user.name "My name"
<li> git config --global user.email my@example.com
<li> .gitignore

```
*.pyc
*~
/myenv
__pycache__
db.sqlite3
/static
.DS_Store
```

<li> git status
<li> git add --all .
<li> git commit -m "commit message"
<li> 깃허브에 레파지토리 만들기
<li> git remote add origin https://github.com/name/repository
<li> git push -u origin master
